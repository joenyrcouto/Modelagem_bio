Introdução

A modelagem de populações estruturadas por idade é uma ferramenta central em demografia e ecologia, permitindo projetar a evolução temporal de uma população a partir de taxas de fertilidade e mortalidade específicas por idade. As matrizes de Leslie, introduzidas por Patrick Leslie na década de 1940, constituem um modelo linear de diferenças que descreve a dinâmica de uma população feminina dividida em classes etárias. Sob a hipótese de que as taxas vitais permanecem constantes, a matriz de Leslie fornece o autovalor dominante, que representa a taxa finita de crescimento da população, e o autovetor dominante normalizado, que dá a distribuição etária estável para a qual a população converge a longo prazo, independentemente da condição inicial. Neste projeto, constrói-se e analisa-se a matriz de Leslie para a população feminina de um país hipotético (“País X”), utilizando dados demográficos realistas de 11 classes etárias de 5 anos, compreendendo idades de 0 a 54 anos. O objetivo é determinar se a população está crescendo, diminuindo ou estável, calcular a distribuição etária estável e verificar numericamente a conjectura de Leslie, que garante a existência de um único autovalor real positivo dominante para matrizes biologicamente plausíveis.

Desenvolvimento

Construção da Matriz de Leslie

A matriz de Leslie \(\mathbf{L}\) é uma matriz quadrada de ordem \(11\), onde a primeira linha contém as taxas de fertilidade \(f_i\) (número médio de filhas nascidas vivas por mulher na classe \(i\) durante um intervalo de 5 anos) e a subdiagonal contém as probabilidades de sobrevivência \(s_i\) (probabilidade de uma mulher na classe \(i\) passar para a classe \(i+1\) após 5 anos). As demais entradas são nulas. Para os dados fornecidos, construiu-se a matriz \(\mathbf{L}\) explicitamente.

Análise do Crescimento a Longo Prazo

Calculam-se todos os autovalores da matriz \(\mathbf{L}\). O autovalor dominante \(\lambda_1\) é aquele com o maior módulo e, para matrizes de Leslie biologicamente plausíveis, é real e positivo. Ele representa a taxa finita de crescimento da população a cada 5 anos: se \(\lambda_1 > 1\), a população cresce; se \(\lambda_1 = 1\), permanece estável; se \(\lambda_1 < 1\), declina.

O autovalor dominante é \(\lambda_1 = 1.0741\) (real positivo). Como \(\lambda_1 > 1\), a população feminina do País X está em crescimento. A taxa de crescimento por período de 5 anos é de \((\lambda_1 - 1) \times 100\% = 7.41\%\).

A Distribuição Etária Estável

O autovetor dominante \(\mathbf{v}_1\) associado a \(\lambda_1\) fornece a estrutura etária relativa. Normalizando-o, obtém-se a distribuição percentual estável \(\mathbf{u}_1 = \mathbf{v}_1 / \sum(\mathbf{v}_1)\). A Tabela 3 apresenta esses valores, e a Fig. 1 mostra o gráfico de barras.

A distribuição etária estável é fortemente inclinada para as idades mais jovens: a classe 0-4 anos concentra aproximadamente 12.94% da população, e as proporções diminuem gradualmente com a idade. Isso é típico de populações em crescimento.

Verificação Numérica da Conjectura de Leslie

A conjectura de Leslie afirma que, para uma matriz de Leslie irredutível e aperiódica, existe um único autovalor real positivo que é estritamente maior em módulo do que todos os outros. Analisando os autovalores obtidos, observa-se que o autovalor dominante é \(\lambda_1 = 1.0741\), com parte imaginária nula, sendo portanto real e positivo. Além disso, os demais autovalores são todos inferiores a 1, confirmando que \(|\lambda_1|\) é estritamente maior que \(|\lambda_j|\) para \(j>1\). Portanto, a conjectura de Leslie é validada numericamente para esta matriz. Biologicamente, essa propriedade garante que, a partir de qualquer distribuição etária inicial não negativa e não nula, a população converge para a estrutura estável dada pelo autovetor dominante, com taxa de crescimento determinada por \(\lambda_1\), independentemente das condições iniciais.

Conclusão

O estudo da matriz de Leslie para a população feminina do País X permitiu quantificar sua dinâmica futura sob a hipótese de constância das taxas de fertilidade e mortalidade. O autovalor dominante \(\lambda_1 = 1.0741\) indica que a população está crescendo a uma taxa de aproximadamente 7.41% a cada cinco anos (crescimento anual médio de cerca de 1.44%). A distribuição etária estável mostra uma população jovem, com 12.94% das mulheres na faixa de 0-4 anos, típica de países em transição demográfica. A verificação numérica da conjectura de Leslie confirmou que \(\lambda_1\) é o único autovalor real positivo e de maior módulo, garantindo a convergência para a estrutura estável independentemente da condição inicial.

Código 1: Construção da matriz de Leslie

Código 
using LinearAlgebra, Markdown
# Dados
fertilidade = [0.0000, 0.0005, 0.0121, 0.1534, 0.3891, 0.4502,
               0.3121, 0.1895, 0.0562, 0.0051, 0.0000]
sobrevivencia = [0.9912, 0.9989, 0.9985, 0.9972, 0.9961, 0.9950,
                 0.9935, 0.9910, 0.9871, 0.9812, 0.9725]
n = length(fertilidade)
# Constroi matriz
L = zeros(n, n)
L[1, :] = fertilidade[:]
for i in 1:n-1
    L[i+1, i] = sobrevivencia[i]
end
# Gera tabela Markdown com a matriz completa (11x11)
# Arredonda para 4 casas decimais
L_rounded = round.(L, digits=4)
# Cabeçalho: Classe 1 a Classe 11
cabecalho = "| " * join(["Classe $i" for i in 1:n], " | ") * " |"
linha_separadora = "|" * join(["---" for _ in 1:n], "|") * "|"
linhas = [cabecalho, linha_separadora]
for i in 1:n
    linha = "| " * join([string(L_rounded[i, j]) for j in 1:n], " | ") * " |"
    push!(linhas, linha)
end
tabela_md = join(linhas, "\n")
display(MIME("text/markdown"), tabela_md)

Código 2: Cálculo dos autovalores

Código 
using LinearAlgebra, Markdown
# Dados (redundante, para código autocontido)
fertilidade = [0.0000, 0.0005, 0.0121, 0.1534, 0.3891, 0.4502,
               0.3121, 0.1895, 0.0562, 0.0051, 0.0000]
sobrevivencia = [0.9912, 0.9989, 0.9985, 0.9972, 0.9961, 0.9950,
                 0.9935, 0.9910, 0.9871, 0.9812, 0.9725]
n = length(fertilidade)
L = zeros(n, n)
L[1, :] = fertilidade[:]
for i in 1:n-1
    L[i+1, i] = sobrevivencia[i]
end
# Autovalores
vals = eigen(L).values
idx = sortperm(abs.(vals), rev=true)
vals_sorted = vals[idx]
# Tabela Markdown com todos os 11 autovalores
linhas = []
push!(linhas, "| Índice | Parte Real | Parte Imag. | Módulo |")
push!(linhas, "|--------|------------|-------------|--------|")
for i in 1:n
    v = vals_sorted[i]
    linha = "| $i | $(round(real(v), digits=4)) | $(round(imag(v), digits=4)) | $(round(abs(v), digits=4)) |"
    push!(linhas, linha)
end
tabela_md = join(linhas, "\n")
display(MIME("text/markdown"), tabela_md)

Código 3: Autovetor dominante e normalização

Código 
using LinearAlgebra, Markdown
# Dados (redundante para auto-continuidade)
fertilidade = [0.0000, 0.0005, 0.0121, 0.1534, 0.3891, 0.4502,
               0.3121, 0.1895, 0.0562, 0.0051, 0.0000]
sobrevivencia = [0.9912, 0.9989, 0.9985, 0.9972, 0.9961, 0.9950,
                 0.9935, 0.9910, 0.9871, 0.9812, 0.9725]
n = length(fertilidade)
# Constrói matriz
L = zeros(n, n)
L[1, :] = fertilidade[:]
for i in 1:n-1
    L[i+1, i] = sobrevivencia[i]
end
# Autovalores e autovetores
vals = eigen(L).values
idx = sortperm(abs.(vals), rev=true)
vecs = eigen(L).vectors
v1 = vecs[:, idx[1]]
u1 = v1 / sum(v1)
porcentagem = real.(100 * u1)   # extrai apenas a parte real
classes = ["0-4", "5-9", "10-14", "15-19", "20-24", "25-29",
           "30-34", "35-39", "40-44", "45-49", "50-54"]
linhas = []
push!(linhas, "| Classe etária | % estável |")
push!(linhas, "|---------------|-----------|")
for i in 1:n
    linha = "| $(classes[i]) | $(round(porcentagem[i], digits=2))% |"
    push!(linhas, linha)
end
tabela_md = join(linhas, "\n")
display(MIME("text/markdown"), tabela_md)
